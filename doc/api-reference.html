    API Reference distributed_simulator v0.1.0

    Modules

    Simulator.BaseConstants

    Constant module to be used inside framework core. Contatins only base constants which are shared between all simulations.

    Simulator.Cell

    Module which should be used by exactly one module in every simulation. That module will be called Cell module.

    Simulator.Constants

    Module which should be used in the Constant module in every simulation. Constant module provides other modules with useful constants (as Nx requires numbers, it is a workaround to make code more readables).

    Simulator.Helpers

    Module containing useful functions (defns).

    Simulator.Metrics

    Module which should be used by exactly one module in every simulation. That module will be called Metrics module.

    Simulator.PlanCreator

    Module which should be used by exactly one module in every simulation. That module will be called PlanCreator module.

    Simulator.PlanResolver

    Module which should be used by exactly one module in every simulation. That module will be called PlanResolver module.

    Simulator.Printer

    Module responsible for saving visualization data and metrics to the files. Additionally contatins functions helpful in debugging.

    Simulator.Simulation

    Entry point. Every simulation should call Simulator.Simulation.start/5.

    Simulator.Types

    Module containing type shared between modules in the Simulator.

    Simulator.WorkerActor

    GenServer responsible for simulating one shard.

    Simulator.WorkerActor.Consequences

    Module contataining a Worker's function responsible for the consequences.

    Simulator.WorkerActor.Plans

    Module contataining Worker's functions responsible for the plans.

    Simulator.WorkerActor.Signal

    Module contataining Worker's functions responsible for the signal.
